用Java实现SQL的嵌套集设计 Nested Set Modified Preorder Tree Traversal

English

前言

最近有一个目录文件入库的需求，条件是：

  1. 入参是解压后的文件夹路径；
  2. 解压后的文件不存在变更及更新的情况。

需求详情是：

  1. 文件夹及其所有子目录和子文件，都需要解析成树结构响应给前端；
  2. 需要在点击每一级目录时，都拿到这个目录下所有文件（包括子目录下的文件）进行一些业务数据的统计；
  3. 只能使用关系型数据库MySQL。
